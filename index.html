Data Notes Actual-value Mode

This page prioritizes data fidelity over visual enhancement. The raster is rendered with nearest-neighbor display, full opacity, and a full-period actual value range by default. The GeoTIFF is expected to be projected to EPSG:32632 / WGS 84 UTM zone 32N before deployment.
